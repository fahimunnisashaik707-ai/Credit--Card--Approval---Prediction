# 4. Project Planning

## Development Modules & Sequence

| # | Module | Description |
|---|---|---|
| 1 | Environment Setup | Python environment, install Flask, joblib, pandas, scikit-learn |
| 2 | Data Collection & Exploration | Load `creditcard_data.csv`, explore in `notebook.ipynb` |
| 3 | Data Preprocessing | Encode categorical fields (Gender, Income Type, Education Level) |
| 4 | Model Training & Evaluation | Train a classification model on historical approval data |
| 5 | Model Persistence | Save trained model as `models/card_model.joblib` |
| 6 | Backend Development | Flask app (`app.py`) — `/` and `/predict` routes |
| 7 | Frontend Development | HTML form (`templates/`) + styling (`static/`) |
| 8 | Testing & Deployment | Local testing via Flask dev server; deployment optional |

## Tools Used
- **Editor:** VS Code / Jupyter Notebook
- **Version Control:** Git & GitHub
- **Language:** Python 3
