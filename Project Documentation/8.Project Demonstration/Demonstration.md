1. Open the web form
2. Enter sample applicant details (e.g. Gender: Male, Income: 75000, Income Type: State/Government Sector, Education: Master's Degree)
3. Submit and show the Approved result with its explanation
4. Try a lower-income profile to show a Rejected result
