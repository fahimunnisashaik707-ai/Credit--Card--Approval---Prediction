# Credit Card Approval Prediction

## Overview
This project focuses on predicting credit card approvals using machine learning techniques. It includes data preprocessing, exploratory data analysis, and a predictive model, exposed via a simple web interface.

## Project Structure
- `app.py`: Main application file for the web interface.
- `notebook.ipynb`: Jupyter notebook containing data exploration and model training.
- `models/`: Directory containing trained machine learning models.
- `data/`: Directory for datasets (Note: `creditcard_data.csv` is ignored by git).
- `static/` & `templates/`: Web application assets (CSS, JS, HTML).
- `Project Documentation/`: Various project phase documentation folders.

## Setup
1. Clone the repository.
2. Install necessary dependencies (e.g., `pip install -r requirements.txt` if available).
3. Place `creditcard_data.csv` in the `data/` directory.
4. Run `python app.py` to start the application.
